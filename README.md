# Data Manage System

Vue + Quasar 打造的类资源管理器的 **资料管理系统**。

[Demo](https://dms.jeremyjone.com/)

> 仅供学习使用。

## 特色

可以学习到多种实用方法。

- 支持直接运行
- TS 编写
- 多种自定义组件
- 简易的后台管理模板
- 拖拽功能
- 切换视图
- 切换皮肤
- 上传环境
- 完整的路由环境
- 丰富的 CSS 样式

## 使用方式

### 下载

```sh
git clone https://github.com/jeremyjone/dms.git
```

### 安装环境

```
yarn install
```

### 启动

```
yarn serve
```

### 打包

```
yarn build
```
